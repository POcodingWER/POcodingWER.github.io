# 빠르게 성장하는 프론트엔드 개발자입니다 👋

**블록체인(Solidity) 개발자로 시작해 3.5년 만에** DAU 12,000명 서비스의 멀티 플랫폼 아키텍처를 설계하고,
AWS 비용 60% 절감(연 240만원)과 로딩 속도 60% 개선을 이끌어낸 프론트엔드 개발자입니다.

## 🚀 핵심 성과

### 📊 비용 최적화 전문성

- **AWS 비용 60% 절감**: CloudFront 최적화로 월 $300 → $120 (연 240만원 절감)
- **CloudFront 요청 90% 감소**: 1,200만 → 100만 요청/월
- **초기 로딩 속도 60% 개선**: 3초 → 1.2초

### 🏗️ 대규모 아키텍처 설계

- **무중단 마이그레이션 설계**: Nuxt SSR → React CSR, 6개월 프로젝트 리드 (2인 팀)
- **모노레포 아키텍처 구축**: 10개 독립 서비스 통합 관리 (charging, conversion, laundry, main, point, ticket 등)
- **Lambda + S3 페이지별 독립 배포**: 서비스 다운타임 0분

### 📱 비즈니스 임팩트

- **DAU 증대 기여**: 세탁 서비스 런칭으로 평균 12,000명 (최대 20,000명) 달성
- **크로스 플랫폼 협업**: 앱-웹 브릿지 통신 구현, 블루투스 IoT 디바이스 연동
- **장기 미해결 이슈 해결**: Sentry 도입 + 2개월 디버깅으로 블루투스 통신 에러 완전 해결

## 💼 경력

### 프론트엔드 개발자

**2022.09 ~ 현재 (3년 4개월)**

캠퍼스 중심 B2C 모바일 앱 개발 회사에서 10개 독립 서비스를 운영하며 기술 전환과 최적화를 주도했습니다.

#### 주요 업무

- **Nuxt.js → React.js 마이그레이션 프로젝트 리드**
  - 10개 핵심 페이지 전환 (main, charging, conversion, laundry, point, ticket 등)
  - Lambda + S3 기반 점진적 배포 전략 설계
  - 2인 팀, 6개월 소요, 서비스 무중단 운영
- **모노레포 아키텍처 구축**

  - pnpm workspace 기반 멀티 서비스 통합
  - 공통 UI 라이브러리 설계 (packages/ui, packages/styles)
  - 페이지별 독립 빌드/배포 파이프라인 구성

- **AWS 인프라 최적화**

  - CloudFront 캐싱 전략 수립으로 비용 60% 절감
  - S3 파일 압축 및 정적 리소스 최적화
  - 로딩 속도 3초 → 1.2초 개선

- **세탁 서비스 개발 및 운영**

  - 블루미터(블루투스 IoT) 연동 세탁기 제어 시스템 개발
  - 앱-웹 브릿지 통신 구현
  - DAU 12,000명 규모 서비스 안정적 운영

- **모니터링 시스템 구축**

  - Sentry 도입으로 실시간 에러 트래킹
  - 2개월 디버깅으로 장기 미해결 블루투스 통신 이슈 해결
  - 사용자 경험 개선 및 이슈 대응 시간 단축

- **개발 문화 개선**
  - 주간 기술 공유 세션 제안 및 진행
  - 반복 케이스 문서화로 지식 공유 시스템 구축
  - Storybook 도입으로 디자이너-개발자 협업 효율화

#### 기술 스택

React, TypeScript, Nuxt.js, Vue.js, Motion.js, Sentry, AWS (Lambda, S3, CloudFront), pnpm workspace, Storybook

### 블록체인 개발자 & NFT 프로젝트

**2022.06 ~ 2022.09 (3개월)**

Solidity 기반 스마트 컨트랙트 개발 및 NFT 민팅 플랫폼 구축 경험을 통해 **블록체인 기술과 Web3 생태계에 대한 깊은 이해**를 갖추게 되었습니다. 이후 프론트엔드 개발로 전환하며 더 넓은 사용자에게 가치를 전달하는 방향으로 커리어를 발전시켰습니다.

**주요 프로젝트:**

- **Delica NFT 민팅 플랫폼**: Solidity 스마트 컨트랙트 설계 및 React 기반 프론트엔드 구현
- **Bellygom NFT 프로젝트**: 이더리움 기반 NFT 발행 시스템 개발

**기술 스택:** Solidity, Web3.js, Ethereum, React, ethers.js

**전환 이유:** NFT 시장 침체 이후, **더 많은 사용자에게 실질적 가치를 제공**하는 프론트엔드 개발로 방향 전환. 블록체인 개발 경험은 복잡한 상태 관리, 비동기 처리, 보안 이슈 대응에 큰 도움이 되었습니다.

## 🎯 강점

### 1. 빠른 기술 전환 능력 ⚡

**Solidity → Vue.js → React → AWS 인프라**까지 3.5년 만에 습득

- 블록체인 개발로 시작했지만 프론트엔드 역량을 빠르게 구축
- 새로운 기술 스택에 빠르게 적응하고 실전 투입 가능
- Web3 경험으로 지갑 연동, 블록체인 통신 등 차별화된 역량 보유

### 2. 비즈니스 임팩트 중심 개발

단순 기능 구현이 아닌 비용 절감, 성능 개선, DAU 증대 등 측정 가능한 성과 창출

### 3. 아키텍처 설계 경험

모노레포, 무중단 배포, 멀티 서비스 운영 등 시니어급 아키텍처 설계 경험 보유

### 4. 문제 해결 능력

2개월 장기 디버깅, 블루투스 통신 이슈 해결 등 복잡한 문제에 대한 근본적 해결 경험
